# 黃曆／奇門／六壬 開源知識庫

## 體積

| 檔案 | 約略大小 |
|------|----------|
| `core-db.json` | ~55 KB |
| `lib/huangli-db.js` | ~55 KB |
| **合計** | **~0.1 MB**（遠低於 500 MB 上限） |

## 內容

- **節氣表**（1980–2040）：定月建、陰陽遁、旺衰
- **黃曆表**：建除宜忌、青龍黃黑道、冲煞、月德天德、彭祖、方位
- **奇門表**：拆補陰陽遁局數、九星八門八神、格局斷語
- **六壬表**：月將、貴人、干寄宮、三傳取法提示
- **個人命理**：1985／1995 日主喜用、十神規則
- **評分權重**：天時綜合分

## 重建

```bash
python3 scripts/build_db.py
```
